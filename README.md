# Android aplikace pro vykazování pracovního času a výpočtu mzdy
Tato aplikace vznikla v rámci individuální semestrální práce předmětu Programování mobilních aplikací na Technické univerzitě v Liberci.
## Použité technologie

* Kotlin
* Firebase

## Základní přehled funkcí aplikace
* Umožňuje vytvářet, upravovat a odstraňovat záznamy.
* Záznamy se ukládají do cloudové databáze Firebase
* Ve výkazech je poté možné hledat a filtrovat
* Lze vypočítat mzdu dle počtu odpracovaných hodin a hodinové sazbě, kterou lze v aplikaci upravovat
